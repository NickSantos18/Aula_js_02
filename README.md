My Hero Academy – Site Interativo

Criei este site inspirado em My Hero Academia usando HTML, CSS e JavaScript.
Nele, é possível escolher classes e individualidades, deixando a experiência interativa.
Usei o HTML para estruturar as páginas, o CSS para o visual e estilização,
e o JavaScript para controlar as escolhas e a interação do usuário.
Esse projeto me ajudou a praticar lógica, organização de código e integração entre as três tecnologias.
